Conéctate directamente a datos del INE desde Tableau con nuestro WDC

¿Qué es un Conector de Datos Web?

Los conectores de datos web (WDC por sus siglas en inglés), son archivos HTML que incluyen código JavaScript y permita a Tableau conectarse a datos que están disponibles a través de Internet.

¿Qué hace este WDC?

Desde The Information Lab España hemos creado un conector que permite consultar directamente desde Tableau datos de series históricas del ine disponibles a través de su servicio web Tempus3.

Tempus3 es el servicio API JSON del INE que permite de este modo acceder mediante peticiones URL a toda la información disponible en INEBase. Puede consultar más información en la web del INE.

Se trata por tanto de un servicio con muchísimas información, que permite consultar miles de datos distintos del INE, si bien el proceso para aquellos que no estén familiarizados con APIs, servicios web y demás no es sencillo. Por esta razón hemos desarrollado este conector de datos que permite realizar consultas directas al INE desde Tableau.

¿Qué consultas están disponibles?

El servicio Tempus3 del INE tiene cientos de posibles combinaciones de datos que se pueden consultar por lo que ofrecer todas las posibles combinaciones no es el objetivo de este WDC. Nuestro objetivo es facilitar el acceso a algunas de las series estadísticas del INE más frecuentes y/o interesantes.

¿Cómo se utiliza?

Simplemente abre Tableau, y en la sección de conectarse a datos, haz click en Conectar > A un servidor > Conector de datos web y pega la siguiente URL:

https://pablosdt.github.io:443/inewdc/

Te aparecerá esta misma web, y un poco más abajo podrás ver un menú desplegable con las consultas disponibles. Selecciona la que quieras consultar (aparecerá una descripción más detallada de dicha consulta) e indica el número de periodos que te interesa analizar (pueden ser días, meses, trimestres o años dependiendo de la frecuencia con que el INE publica dicha fuente de datos), y haz click en el botón Obtener datos. El WDC se conectará los datos más recientes del INE y los descargará en un estracto de Tableau.
